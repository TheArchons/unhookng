# UnhookNG

![Screenshot of YouTube with the extension open](https://github.com/TheArchons/unhookng/blob/main/screenshots/s2.png)

A browser extension that modifies YouTube to make it less distracting and addictive, including hiding recommended videos, shorts, homepage recommendations, and much more. Everything is editable through the popup menu. This fixes many issues from the original Unhook.

Full Feature list:

- Hide Home Feed
- Redirect To Subscriptions
- Hide Video Sidebar
- Hide Recommended
- Hide Live Chat
- Hide Playlist
- Hide Fundraiser
- Hide End Screen Cards
- Hide Shorts
- Hide Comments
- Hide Profile Photos
- Hide Mixes
- Hide Merch
- Hide Video Info
- Hide Buttons Bar
- Hide Channel
- Hide Description
- Hide Top Header
- Hide Notifications
- Hide Explore
- Hide More from YouTube
- Hide Subscriptions
