# UnhookNG

![Promo Tile: Unhook, Remove YouTube Distractions](https://github.com/TheArchons/unhookng/blob/main/screenshots/promo_tile.png?raw=true)

A browser extension that modifies YouTube to make it less distracting and addictive, including hiding recommended videos, shorts, homepage recommendations, and much more. Everything is editable through the popup menu. This fixes many issues from the original Unhook.

## Install

Chrome: https://chromewebstore.google.com/detail/unhook-ng/ebpdaohfgimfpokmmollfkcabmhkcpah

Firefox: https://addons.mozilla.org/en-US/firefox/addon/unhook-ng/

The ZIP file is also available in the [Releases](https://github.com/TheArchons/unhookng/releases) tab.

## Features

- Hide Home Feed
- Redirect To Subscriptions
- Hide Video Sidebar
- Hide Recommended
- Hide Live Chat
- Hide Playlist
- Hide Fundraiser
- Hide End Screen Cards
- Hide Shorts
- Hide Comments
- Hide Profile Photos
- Hide Mixes
- Hide Merch
- Hide Video Info
- Hide Buttons Bar
- Hide Channel
- Hide Description
- Hide Top Header
- Hide Notifications
- Hide Explore
- Hide More from YouTube
- Hide Subscriptions
